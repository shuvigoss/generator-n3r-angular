generator-n3r-angular
=====================

N3 angular yeoman generator


安装
=====================
    我已将该模块提交到npm registry 
[查看](https://registry.npmjs.org/generator-n3r-angular)<br />
    npm install generator-n3r-angular -g
> 创建一个文件夹，随意地方
    mkdir test
> 会要求输入工程名称，根据情况输入
    yo n3r-angular
> 会安装app所需包
    bower install
